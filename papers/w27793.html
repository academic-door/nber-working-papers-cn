An Optimal Test for Strategic Interaction in Social and Economic Network Formation between Heterogeneous Agents

异质性主体间社会与经济网络形成中战略互动的优化检验

Andrin Pelican, Bryan S. Graham

理论与方法 未指明区域

Abstract

Consider a setting where N players, partitioned into K observable types, form a directed network. Agents' preferences over the form of the network consist of an arbitrary network benefit function (e.g., agents may have preferences over their network centrality) and a private component which is additively separable in own links. This latter component allows for unobserved heterogeneity in the costs of sending and receiving links across agents (respectively out- and in- degree heterogeneity) as well as homophily/heterophily across the K types of agents. In contrast, the network benefit function allows agents' preferences over links to vary with the presence or absence of links elsewhere in the network (and hence with the link formation behavior of their peers). In the null model which excludes the network benefit function, links form independently across dyads in the manner described by Charbonneau (2017). Under the alternative there is interdependence across linking decisions (i.e., strategic interaction). We show how to test the null with power optimized in specific directions. These alternative directions include many common models of strategic network formation (e.g., “connections” models, “structural hole” models etc.). Our random utility specification induces an exponential family structure under the null which we exploit to construct a similar test which exactly controls size (despite the null being a composite one with many nuisance parameters). We further show how to construct locally best tests for specific alternatives without making any assumptions about equilibrium selection. To make our tests feasible we introduce a new MCMC algorithm for simulating the null distributions of our test statistics.

中文摘要

考虑一个场景，其中N个参与者被划分为K个可观测类型，形成一个有向网络。主体对网络形式的偏好由一个任意的网络收益函数（例如，主体可能偏好其网络中心性）和一个在自身链接上可加分离的私人部分组成。后一部分允许在发送和接收链接的成本方面存在未观测到的异质性（分别是出度和入度异质性），以及跨K类主体的同质性/异质性。相比之下，网络收益函数允许主体对链接的偏好随网络中其他地方链接的存在或缺失而变化（因此也随其同伴的链接形成行为而变化）。在排除网络收益函数的零模型中，链接按照Charbonneau（2017）描述的方式在二元组之间独立形成。在备择假设下，链接决策之间存在相互依赖（即战略互动）。我们展示了如何在特定方向上优化检验功效来检验零假设。这些备择方向包括许多常见的战略网络形成模型（例如，“连接”模型、“结构洞”模型等）。我们的随机效用设定在零假设下诱导出一个指数族结构，我们利用该结构构建了一个相似检验，该检验精确控制了检验尺度（尽管零假设是一个包含许多冗余参数的复合假设）。我们进一步展示了如何在不假设均衡选择的情况下，为特定备择假设构建局部最优检验。为了使我们的检验可行，我们引入了一种新的MCMC算法来模拟检验统计量的零分布。
